## Setup


```bash
git clone git@github.com:MikRyam/test-cifra-b.git
cd test-cifra-b

npm install
# or
yarn install
```

## Getting Started

```bash
npm run start
# or
yarn start
```

```bash
# run Prettier
npm run format

```

## Bundle application

```bash
# run build
```
